# ملاحظة جاهزية Hotelbeds Transfers للعميل

## الحالة الحالية

مسار العميل لخدمة Hotelbeds Transfers جاهز للاختبار الداخلي فقط:

- البحث يعمل عبر Hotelbeds Transfers.
- صفحة اختيار/checkout تعمل.
- صفحة التأكيد تعمل.
- مكون الفاتشر جاهز للعرض والطباعة.
- لا يوجد حجز تلقائي.
- لا يتم عرض raw JSON للعميل.
- لا يظهر `rateKey` كاملًا للعميل.
- صفحة الشهادة بقيت داخلية وليست جزءًا من تجربة العميل العامة.

## شرط الإنتاج

لا يجب تنفيذ أي حجز Hotelbeds Transfers في الإنتاج قبل نجاح الدفع.

في الإنتاج يجب أن يتم استدعاء تأكيد Hotelbeds Transfers فقط بعد حدث Stripe:

`payment_succeeded`

ويجب إبقاء:

`HOTELBEDS_TRANSFERS_BOOKING_ENABLED=false`

إلى أن يتم ربط الدفع بشكل نهائي، واستخدام credentials الإنتاج المعتمدة، ومراجعة flow الحجز بعد الدفع.

## ملاحظات أمان

- لا يتم وضع أي API key أو secret داخل الكود أو التوثيق.
- لا يتم طباعة headers أو signature أو أسرار في logs.
- لا يتم تفعيل Stripe live ضمن هذا التجهيز.
- هذا التجهيز ليس حجزًا إنتاجيًا قبل ربط الدفع والاعتماد النهائي.
